# Milestone 00: Project preparation

## Goal

Prepare the Vite project so the portfolio scene architecture can be implemented and deployed as a static site.

## Implementation scope

- Confirm Vite, React, and TypeScript are configured.
- Configure Tailwind CSS through the Vite plugin.
- Configure Motion for React, linting, formatting, Vitest, and Playwright.
- Establish directories for semantic scenes, navigation, transitions, assets, and tests.
- Confirm the project runs and builds from a clean checkout.

## Explicit non-goals

- Do not add real portfolio copy.
- Do not implement authored intro choreography.
- Do not add a backend, API route, or runtime data dependency.
- Do not add a horizontal document or scrolling library.

## Acceptance criteria

- TypeScript, linting, unit tests, and browser tests have runnable commands.
- A clean checkout can install dependencies and produce `dist/`.
- The base document has no unexpected persistent overflow.
- Placeholder content is not presented as final portfolio content.

## Exit condition

The Vite project builds successfully and the planned seams exist without real portfolio content or authored timeline work.
